Przygotowanie do naprawy dysku twardego:

  • wyłączamy wszystkie procesy oraz programy
  • jeśli posiadamy zaszyfrowane partycje to je odszyfruwujemy

    Programy do naprawy dysku:

  • Victoria (diagnoza i naprawa dysku)
  • HDTune (diagnoza i naprawa dysku)
  • HDTune PRO (diagnoza i naprawa dysku)
  • TestDisk (zagubione partycje)
  • PC Inspector File Recvery (Odzyskiwanie plików)
  • HDD Tune (benchmark)
  • HDD Tune PRO (benchmark)
  • HDtach (benchmark)

    Kolory oraz ich znaczenie:

  • Zielony, szary - sektory są dobre
  • Pomarańczowy - prawidłowe, aczkolwiek mogące ukazać sie wadliwe
  • Czerwony - "bad sectory"

    Objawy wolnego działania dysku:

  • Wolny komputer
  • Wolne kopiowanie
  • Słaby wynik bechmarku

    • Przyczyny wolnego działania dysku:

  • Błędy systemu
  • Błędy przy przesyłaniu UDMA
  • Wina podłączenia dysku

    • Rozwiązanie wolnego działania komputera:

  • Reset
  • Ustawienie trybu UDMA
  • Usunięcie kontrolerów
  • Zmiana przewodu dysku
  • Flashback BIOS płyty głównej

SMART - system moniturujący działanie dysku i powiadumujący o jego błędnym działaniu

    Błędy SMART:

  • PW - wkazuje warunek przed awaryjny
  • OC - oznacza oblicznie wartości podczas pracy dysku
  • PE - oznacza pogorszenie spowodowane zużyciem lub wiekiem dysku
  • ER - oznacza parametr częstoczliwości pomiaru błędu
  • EC - oznacza parametr ilości zamierzonych błędów
  • SP - parametr automatycznie ustawiany podczas testu SMART

  • Raw Read Error Rate - parametr określający błąd krytyczny dysku twardego
  • Thrughput Performance - ogólna sprawność dysku
  • Spin Up Time - średni czas na rozpędzenie dysku
  • Start/Stop Count - określa liczbę start dysku oraz jego stopów (zatrzymań)
  • Reallocated Sectors Count - ilość relokowanych sektorów
  • Read Channel Margin - Rezerwa kanału podczas odczytu danych
  • Seek Error Rate - częstoczliwość błędów głowic magnetycznych
  • Seek time Performance - przeciętna wydajność operacji wyszukiwania głowic magnetycznych
  • Power-On Time - ilość godzin w stanie zasilania
  • Spin Retry Count - ilość ponowień prób rozpędzenia dysku
  • Recalibration Retries - liczba żądań rekalibracji
  • Device Power Cycle Count - określa ilość pełnych cykli zasilania dysku
  • Soft Read Error rate - ilość programowych błędów odczytu
  • G-Sense Eroor Rate - częstoczliwość występowania błędów spowodowanych wstrząsami
  • Power Cycle Count - określa ile razy dysk został wyłączony
  • Load/Unload cycle Count - ilość cykli parkowania oraz wyparkowania ze stefy parkowania
  • Temp. - temperatura dysku twardego
  • Reallocation Event Count - ilość operacji remapowania
  • Current Pending Sector Count - określa liczbę niestabilnych sektorów
  • Off-line Uncorrectable Sector Count - ilość niekorygowanych błędów
  • UltraDMC CRC Error Count - łączna ilość błędów CRC w trybie UltraDMA
  • Write Error Rate (Multi Zone Error Rate) - częstość błędów zapisu (Wartość 0 wskazuje, że dysk jest sprawny)

  • Disk Shift - przesunięcie dysków od osi
  • Loaded Hours - czas działania akuratora
  • Load/Unload Retry Count - liczba użyć akuratora głowic magnetycznych
  • Load Friction - Użycie akuratora głowic magnetycznych spowodowane tarciem mechanicznych części dysku
  • Load-in time - łączny czas działania akuratora głowic
  • Torque Amplification Count - ilość prób rozkręcania
  • GMR Head Amplitude - Amplituda drgań głowic podczas pracy
  • Head Flying Hours - Czas w jakim głowica jest pozycjonowana
  • Read Error Retry Rate - częstoczliwość występowania błędów podczas odczytu
